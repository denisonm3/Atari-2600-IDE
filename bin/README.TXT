==========================================================

Atari 2600 IDE v1.0
==========================================================



This is a simple IDE for atari 2600, its intention is to be plug and play.
The current version was only avaliable in Windows, for the assembly 6502.
When you start the application it will create the necessary files.

To run the IDE execute the .jar file, type the following:



java -jar Atari2600IDE.jar
